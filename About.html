Über das MitschriebWiki

Hallo geehrter Besucher. Auf diesen Seiten finden Sie Skripte (und Teil-Skripte) von Vorlesungen, die in Gemeinschaftsarbeit entstehen. Mitarbeit ist dabei erwünscht!

Dieses Projekt wurde von Joachim Breitner (mail@joachim-breitner.der) unter dem Namen lkwiki zum Wintersemester 04/05 gestartet.

Zum Wintersemester 06/07 wurde es auf eine neue Softwarebasis gestellt, der Software latexki: https://github.com/nomeata/latexki

Das MitschriebWiki bekam Mitte 2012 ein Redesign (siehe Ist-Zustand), erstellt von http://www.breitnerundbreitner.de/ verpasst, und wurde deutlich übersichtlicher und schöner. Kommentare zum Layout gerne an Thomas (mail@breitnerundbreitner.de).

Mitte 2016 zog das Mitschriebwiki von Joachims Server auf Github http://github.com/nomeata/mitschriebwiki; Änderungen können jetzt per Pull-Request vorgenommen werden. Dadurch wird sichergestellt, dass die Inhalte noch lange verfügbar bleiben, auch wenn Joachim seinen Server irgendwann einstampfen sollte.
